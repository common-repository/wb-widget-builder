Wb Widget Builder, Build your new Widget easily from your wordpress admin page without creating php files.


== Description ==

Wb Widget Builder plugin is a Wordpress Plugin that makes it very simple and easy to create a new custom widgets. With the help of Wb Widget Builder user can create multiple widgets without creating Widget classes and without touching php files. It'll let you focus on your widget input and output rather than creating whole class files, including and initilazing widgets etc. 
 


== Installation ==

1. Visit Plugins > Add New
2. Search for “Wb Widget Builder”
3. Activate Wb Widget Builder from your Plugins page
4. Click on Widget Builder menu at left menu bar
5. Add new widget and publish
6. Go to Appearance > Widgets 
7. Find our new widget in left available widgets and add it to your sidebar.



== Frequently Asked Questions ==

= How to use Wb Widget Builder? =

    1. Click on Widget Builder menu at left menu bar and Create a new Widget.
    2. Add Form code, update code, widget code and helper functions if needed.
    3. publish it
    4. go to the widgets page your new widget should already be there.
    5. add your widget to sidebar and save it.
